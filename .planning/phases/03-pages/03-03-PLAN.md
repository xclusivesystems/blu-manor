- Each item renders as:
  - Container: `rounded-xl border border-border bg-bg-card`
  - Button trigger: `<button>` element (NOT div) with `w-full flex items-center justify-between px-6 py-4 text-left cursor-pointer`
  - Question text: `font-medium text-foreground`
  - ChevronDown icon from Lucide: rotates 180deg when open, using `transition-[transform] duration-200` (NOT transition-all)
