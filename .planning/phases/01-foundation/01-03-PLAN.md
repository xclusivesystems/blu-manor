  <action>
    **AnimatedSection.tsx** — "use client" component:
    - Import `motion` from `framer-motion`, `useReducedMotion` hook from `framer-motion`
    - Props: `children: ReactNode`, `delay?: number` (default 0), `className?: string`
    - Use `useReducedMotion()` to conditionally disable `y` transform: `initial={{ opacity: 0, y: prefersReduced ? 0 : 24 }}`
    - `whileInView={{ opacity: 1, y: 0 }}`, `viewport={{ once: true, margin: "-80px" }}`
    - `transition={{ duration: 0.5, delay, ease: "easeOut" }}`
    - Per MASTER.md: respect `prefers-reduced-motion: reduce` — disable transforms, use opacity only

    **Button.tsx** — "use client" component (for potential onClick handlers):
    - Import `cn` from `@/lib/utils`, `Link` from `next/link`
    - Props: `variant?: "primary" | "secondary" | "ghost"` (default "primary"), `size?: "sm" | "md" | "lg"` (default "md"), `href?: string`, `children: ReactNode`, `className?: string`, plus standard button HTML attributes
    - If `href` provided, render as `<Link>`. Otherwise render as `<button>`.
    - Variant classes per MASTER.md:
      - primary: `bg-primary text-bg-deep font-semibold hover:bg-primary-light hover:-translate-y-0.5 transition-[color,background-color,transform] duration-200`
      - secondary: `border-2 border-primary text-primary hover:bg-primary-dim transition-[color,background-color,border-color] duration-200`
      - ghost: `text-muted hover:text-foreground transition-[color] duration-200`
    - Size classes: sm (`px-4 py-2 text-sm`), md (`px-6 py-3`), lg (`px-8 py-4 text-lg`)
    - Base: `rounded-lg font-medium cursor-pointer inline-flex items-center justify-center`

    **Card.tsx** — server component:
    - Import `cn` from `@/lib/utils`
    - Props: `children: ReactNode`, `hover?: boolean` (default true), `className?: string`
    - Base: `rounded-xl bg-bg-card border border-border p-6 shadow-[0_4px_24px_rgba(15,22,35,0.4)]`
    - Hover (when `hover` prop is true): `hover:border-border-hover hover:shadow-[0_8px_32px_rgba(201,168,76,0.08)] hover:-translate-y-1 transition-[border-color,box-shadow,transform] duration-300`
    - Never use `transition-all`

    **SectionHeader.tsx** — server component:
    - Import `cn` from `@/lib/utils`
    - Props: `label?: string`, `title: string`, `description?: string`, `centered?: boolean` (default true), `className?: string`
    - Label: `text-sm uppercase tracking-wide text-primary font-medium` (Source Sans 3)
    - Title: `text-3xl md:text-4xl font-bold text-foreground font-[family-name:var(--font-libre)] tracking-tight mt-2`
    - Description: `text-muted text-lg mt-4 max-w-2xl` (centered if `centered` prop)
    - Centered layout when `centered` is true: `text-center` with description `mx-auto`
  </action>
