
    4. Delete the default `app/layout.tsx` generated by create-next-app — route groups will provide their own document shells (see Task 2).

    7. Update `next.config.ts` to add `cdn.sanity.io` to `images.remotePatterns` (pre-wired for Phase 2).
    8. Install Phase 1 dependencies: `npm install framer-motion lucide-react`
    9. Copy the logo from the static site to `public/img/logo.png`. If static site path not available, create the `public/img/` directory as placeholder.

    **Anti-patterns to avoid:**
    - Never use `transition-all` anywhere — specify exact properties
    - Never keep `tailwind.config.ts` — Tailwind 4 doesn't use it
    - Never use Google Fonts `<link>` tags — next/font handles this
